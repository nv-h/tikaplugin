S.Kohno様のxdocdiff WinMerge Plugin(http://freemind.s57.xrea.com/xdocdiffPlugin/)を、
WinMergeソースコード内のPluginsにあるDisplayBinaryFilesを基に64bit WinMergeで動作するようにしたものです。

ビルドにはVisual Studio Express 2012 for Windows DesktopとWindows Server 2003 SP1 Platform SDKを使用しています。
ライセンスはWinMergeがGPL2であるためGPL2とします。
